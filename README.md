# Cloudfide Recruitment Task - Junior Data Engineer

### Author: Michał Pędziwiatr

This repository contains my implementation of the `add_virtual_column` function for the recruitment process at Cloudfide.

## Overview

The task was to create a function that adds a new column to a pandas DataFrame by performing math operation (`+`, `-`, `*`) on two already existing columns, based on a provided string input.

## Tech Stack

- **Python 3.x**
- **Pandas**
